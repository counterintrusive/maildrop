Overview

Maildrop+ provides a simple REST API for generating disposable email addresses and receiving messages programmatically. You need an API key to authenticate requests.

Note: All API endpoints require the Authorization: Bearer <your_api_key> header. The web UI uses session cookies instead — you only need the API key for programmatic access.

Authentication

Getting an API key

Create an account — an API key is automatically generated for you on registration. You can view and regenerate your key from the settings panel (gear icon) in the inbox view.

Using your API key

Include your key in the Authorization header on every request:

Authorization: Bearer md_abc123def456...789

Endpoints

GET /get_random_address
Generate a new disposable email address.
Returns a JSON object with the generated email and the domain used.

Example

curl -H "Authorization: Bearer md_your_api_key" \
  https://your-server.com/get_random_address

Response

{
  "email": "james.smith.4821@example.com",
  "domain": "example.com"
}
GET /get_inbox?address=<email>
Retrieve all messages for a given address.
Query parameter: address — the full email address (e.g. james.smith.4821@example.com).
Returns an array of email objects, each with id, from, to, subject, body, and timestamp.

Example

curl -H "Authorization: Bearer md_your_api_key" \
  "https://your-server.com/get_inbox?address=james.smith.4821@example.com"

Response

[
  {
    "id": 1,
    "from": "someone@external.com",
    "to": "james.smith.4821@example.com",
    "subject": "Welcome!",
    "body": "Thanks for signing up...",
    "timestamp": 1717200000
  }
]
POST /send_email
Send an email from a Maildrop+ address.
Sends from one Maildrop+ address to another. Both addresses must use one of the configured domains.

Request body (JSON)

FieldTypeDescription
FromstringSender address on a configured domain
TostringRecipient address on a configured domain
SubjectstringEmail subject line
BodystringEmail body text

Example

curl -X POST -H "Authorization: Bearer md_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "From": "sender@example.com",
    "To": "recipient@example.com",
    "Subject": "Hello",
    "Body": "This is a test message"
  }' \
  https://your-server.com/send_email

Response

{
  "message": "Email sent successfully"
}

Python Example

import requests

API_KEY = "md_your_api_key"
BASE_URL = "https://your-server.com"

headers = {"Authorization": f"Bearer {API_KEY}"}

# Generate a new address
r = requests.get(f"{BASE_URL}/get_random_address", headers=headers)
data = r.json()
print("Email:", data["email"])

# Check inbox
inbox = requests.get(
    f"{BASE_URL}/get_inbox",
    params={"address": data["email"]},
    headers=headers
)
for msg in inbox.json():
    print(f"Subject: {msg['subject']}")
    print(f"From: {msg['from']}")
    print(f"Body: {msg['body']}")
